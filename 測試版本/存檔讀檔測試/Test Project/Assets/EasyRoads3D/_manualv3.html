EasyRoads3D V3 Beta Manual
 

 

User Guide

This user guide explains all the EasyRoads3D features. It is divided in 3 main sections: getting started, EasyRoads3D hierachy, creating roads and dealing with intersections. Use the troubleshooting and known issues sections in the case in you are experiencing unexpected behaviour.

It is especially recommended to carefully read the the second section about the EasyRoads3D hierarchy in the Unity scenes.
